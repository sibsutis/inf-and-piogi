# Справочная информация

### **Объявление массивов**
Объявление массивов может выглядеть следующим образом:
```csharp
int[] mas = new int[10];      // одномерный массив на 10 элементов
int[,] mas2D = new int[3,3]; // двумерный массив размером 3x3
```

### **Инициализация массивов**
Можно инициализировать массивы при объявлении:
```csharp
int[] m1 = new int[5] { 1, 2, 3, 4, 5 }; // массив из 5 целых чисел
float[] m2 = { 1.0f, 2.0f, 3.0f };      // массив из 3 вещественных чисел
int[,] m2d = { {1, 2}, {3, 4}, {5, 6} }; // объявление и инициализация двумерного массива
```

Либо используя циклы:
```csharp
int[] m = new int[10]; // массив целых чисел из 10 элементов
for (int i = 0; i < 10; i++) // заполнение числами от 1 до 10 (индексы от 0 до 9)
{
    m[i] = i + 1;
}

int[,] m2d = new int[3,3]; // массив целых чисел из 9 элементов
// заполнение строк массива числами (например, 1 2 3 / 1 2 3 / 1 2 3)
for (int i = 0; i < 3; i++) 
{
    for (int j = 0; j < 3; j++)
    {
        m2d[i,j] = j + 1;
    }
}
```

### **Вывод значений массива**
Вывод значений массива на экран можно реализовать при помощи цикла `for`:
```csharp
int[] m = { 1, 2, 3, 4, 5 };
for (int i = 0; i < m.Length; i++)
{
    Console.Write("{0} ", m[i]); // вывод на экран элемента массива и пробела
}
```

### **Вывод матрицы в консоль**
```csharp
// Выводим матрицу на консоль
Console.WriteLine("Матрица:");
for (int i = 0; i < rows; i++)
{
    for (int j = 0; j < cols; j++)
    {
        Console.Write(matrix[i, j] + " ");
    }
    Console.WriteLine(); // Переход на новую строку
}
```

### **Работа со строками**
Строка может быть объявлена и инициализирована следующим образом:
```csharp
string str;             // создание строковой переменной
str = "Hello World";    // запись текста в строковую переменную
```

Считать строку из консоли можно при помощи метода `ReadLine`:```csharp
string str;
str = Console.ReadLine(); // создание строковой переменной и запись в нее текста из консоли
```
В сущности, строка является массивом символов, поэтому к ней применимы операции поэлементного доступа:
```csharp
Console.WriteLine(str);     // вывод на экран всей строки
Console.WriteLine(str[3]);  // вывод на экран 4-го символа строки
```

Так же, к строке можно добавлять символы. Например, функцию создания инвертированной версии строки можно реализовать так:
```csharp
string str;
str = Console.ReadLine(); // запись текста в строковую переменную
string strInv = "";       // создание пустой строки

// цикл, начинающийся в конце строки и идущий к её началу
for(int i = str.Length-1; i >= 0; i--) 
{
    strInv += str[i]; // добавление символа к пустой строке
}
```
