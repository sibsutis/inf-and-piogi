# Введение в консольный ввод-вывод в `C#`

Консольные приложения - фундаментальная часть программирования. Они предоставляют текстовый интерфейс для взаимодействия с пользователем, что делает их отправной точкой для изучения нового языка. В C# основой консольного ввода-вывода является класс `System.Console`, который предлагает простые способы для отображения информации и чтения пользовательского ввода.

## Вывод информации в консоль

Для отображения информации в консольном приложении используются методы `Console.Write` и `Console.WriteLine`.

Значение для вывода в консоль передается в функцию `Write/WriteLine` в качестве **аргумента** этой функции.
### Console.Write

Этот метод выводит указанное значение в консоль. Курсор (в нашем случае не курсор мыши, а курсор текста) при этом остается на той же строке, что позволяет выводить несколько значений подряд.

```csharp
// Использование Console.Write
Console.Write("Привет, ");
Console.Write("Мир!");
// Вывод: Привет, Мир!
```
### Console.WriteLine

Этот метод также выводит значение в консоль, но добавляет в конце символ новой строки. Это перемещает курсор на следующую строку, используется для отображения сообщений на отдельных строках.

```csharp
// Использование Console.WriteLine
Console.WriteLine("Это первая строка.");
Console.WriteLine("Это вторая строка.");
// Вывод: 
// Это первая строка.
// Это вторая строка.
```

### Современное форматирование с интерполяцией строк

Хотя существуют и более старые способы форматирования строк, современным и рекомендуемым подходом в `C#` является **интерполяция строк**. Она позволяет встраивать переменные и выражения прямо в строку. Для этого нужно поставить перед строкой знак `$` и заключить переменные в фигурные скобки `{}`.

Вместо

```csharp
string name = "Алиса";
int age = 30;

// Простое сложение (конкатенация) строк. 
Console.WriteLine("Привет, " + name + "! Тебе " + age.ToString() + "лет.");
// Вывод: Привет, Алиса! Тебе 30 лет.
// Мы должны привести age к типу данных строка, так как IDE выдаст ошибку о несоотвествии типов.
```

Используется

```csharp
string name = "Алиса";
int age = 30;

// Использование интерполяции строк
Console.WriteLine($"Привет, {name}! Тебе {age} лет.");
// Вывод: Привет, Алиса! Тебе 30 лет.
// При интерполяции числовые переменные автоматически преобразуются в строковые.
```


## Чтение ввода из консоли

### Console.ReadLine

Самый распространенный метод для получения пользовательского ввода. Он считывает строку текста из консоли и возвращает ее в виде строки (`string`).

```csharp
Console.WriteLine("Пожалуйста, введите ваше имя:"); // Выводим строку в консоль и переносим курсор на следующую строку.
string userName = Console.ReadLine(); // Считываем значение и переносим курсор на новую строку.
Console.WriteLine($"Привет, {userName}!");
```

### Console.Read

Этот метод считывает один символ из потока ввода и возвращает его целочисленный код (или -1, если символов больше нет). Он реже используется для общего ввода.

```csharp
Console.WriteLine("Введите символ:");
int charCode = Console.Read();
Console.WriteLine($"Код символа: {charCode}");
```

### Console.ReadKey

Этот метод ожидает, пока пользователь нажмет любую клавишу, и возвращает объект ConsoleKeyInfo, который содержит информацию о нажатой клавише. Это может использоваться для создания интерактивных меню или реагирования на отдельные нажатия клавиш.

```csharp
Console.WriteLine("Нажмите любую клавишу для продолжения...");
ConsoleKeyInfo keyInfo = Console.ReadKey();
Console.WriteLine($"Вы нажали клавишу {keyInfo.Key}.");
```
