**Задания**
Разработать и реализовать следующие функции и методы:

1. Две функции Max2(int a, int b) и Min2(int a, int b), возвращающие максимум и минимум из двух целых чисел.
2. Две перегруженные версии метода GetStats, который анализирует произвольный набор чисел (целых и вещественных).
3. Метод Swap(ref int a, ref int b), который принимает значения по ссылке и меняет их местами.
4. Метод Reverse(int[] array), который инвертирует порядок элементов в массиве, изменяя его "на месте".
5. Метод Reverse(int[] array), который возвращает новый массив с элементами в обратном порядке, не изменяя исходный массив.
6. Метод TryFind(int[] array, int value, out int index), который выполняет безопасный поиск элемента в массиве и возвращает результат через bool и out-параметр.
7. Универсальный метод Filter для фильтрации массивов, который использует делегаты для применения любого заданного критерия.
8. Два метода (BrokenSafeAppend и RefSafeAppend) для добавления значения в конец массива, чтобы продемонстрировать разницу между передачей массива по значению и по ссылке (с помощью ref).
9. Подготовить ответы на контрольные вопросы, объясняющие разницу в поведении методов из предыдущего пункта.
