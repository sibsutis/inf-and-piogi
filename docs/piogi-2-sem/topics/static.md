
## Определение

**Static в C#** — это **ключевое слово, которое используется для создания статических членов класса**, которые принадлежат самому классу, а не конкретным экземплярам этого класса. 

Статические поля хранят состояние всего класса / структуры. Статическое поле определяется как и обычное, только перед типом поля указывается ключевое слово `static`.
