### Пример использования `Menu` и `MenuItem`

```xml
    <Grid>
        <DockPanel VerticalAlignment="Top">
            <Menu Height="20">
                <MenuItem Header="Комната">
                    <MenuItem Header="Новая комната" Click="Ваш_обработчик"/>
                    <MenuItem Header="Редактировать выбранную"/>
                    <MenuItem Header="Удалить выбранную"/>
                </MenuItem>
            </Menu>
        </DockPanel>

		<!--
        Весь остальной контент
        -->

    </Grid>
```

### **Разбор кода:**

1. **`<Grid>`**: Это корневой элемент макета. `Grid` позволяет располагать элементы в строках и столбцах. В данном случае, он служит контейнером для всего содержимого, включая меню.
    
2. **`<DockPanel VerticalAlignment="Top">`**: `DockPanel` - это контейнер, который позволяет располагать элементы по краям (верх, низ, лево, право). `VerticalAlignment="Top"` указывает, что содержимое `DockPanel` должно быть привязано к верхней части `Grid`.
    
3. **`<Menu Height="20">`**: Это сам элемент меню. `Height="20"` задает высоту меню в 20 пикселей. `Menu` автоматически создает визуальное представление меню с выпадающими подменю.
    
4. **`<MenuItem Header="Комната">`**: Это главный пункт меню, который называется "Комната". `Header` определяет текст, который отображается для этого пункта меню. Когда пользователь наводит курсор на этот пункт, появится выпадающее меню с подпунктами.
    
5. **`<MenuItem Header="Новая комната" Click="Ваш_обработчик"/>`**: Это подпункт меню, который называется "Новая комната". `Header` определяет текст, который отображается для этого пункта меню. `Click="Ваш_обработчик"` указывает, что при нажатии на этот пункт меню должен быть вызван обработчик события `Ваш_обработчик`. Этот обработчик должен быть определен в коде вашего приложения (например, в файле .cs). Он будет отвечать за создание новой комнаты.
    
6. **`<MenuItem Header="Редактировать выбранную"/>`**: Это подпункт меню, который называется "Редактировать выбранную". При нажатии на этот пункт меню, скорее всего, будет открыто окно или диалоговое окно для редактирования выбранной комнаты. В этом примере нет обработчика `Click`, поэтому при нажатии ничего не произойдет.
    
7. **`<MenuItem Header="Удалить выбранную"/>`**: Это подпункт меню, который называется "Удалить выбранную". При нажатии на этот пункт меню, скорее всего, будет удалена выбранная комната. В этом примере также нет обработчика `Click`, поэтому при нажатии ничего не произойдет.
    
8. **`<!-- Весь остальной контент -->`**: Это комментарий, указывающий на место, где будет располагаться остальной контент вашего приложения.

### Пример с несколькими заголовками меню:

```xml
 <Menu Height="20">
     <MenuItem Header="Комната">
         <MenuItem Header="Новая комната"/>
         <MenuItem Header="Редактировать выбранную"/>
         <MenuItem Header="Удалить выбранную"/>
     </MenuItem>
     <MenuItem Header="Второй пункт меню">
         <MenuItem Header="Заглушка"/>

     </MenuItem>

     <MenuItem Header="Третий пункт меню" Click="Еще_один_ваш_обработчик"/>

 </Menu>
```
