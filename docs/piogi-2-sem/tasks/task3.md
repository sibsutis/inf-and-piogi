# Лабораторная работа 3

## Краткое описание

Превращение строк в карточки с использованием DataTemplate и Binding. Реализация связи между списком и панелью деталей.

---

## Обязательные требования

### 1. DataTemplate для карточки
- [ ] Создан `DataTemplate` для отображения фильма
- [ ] Карточка содержит: постер, название, длительность
- [ ] Шаблон вынесен в `Window.Resources`

### 2. Загрузка изображений
- [ ] Постеры загружаются по URL из JSON
- [ ] Используется элемент `Image` с `Binding`

### 3. Панель деталей
- [ ] Справа отображается детальная информация о выбранном фильме
- [ ] Показаны: название, большой постер, описание, длительность

### 4. Binding SelectedItem
- [ ] При выборе фильма в списке — обновляется панель деталей
- [ ] Используется `Binding SelectedItem` или `SelectionChanged`

### 5. Нет хардкода
- [ ] Все данные приходят с API
- [ ] Нет захардкоженных строк в XAML

## Критерий сдачи

1. Запускается приложение
2. Список отображает карточки с постерами (не просто текст)
3. Клик по карточке обновляет панель деталей справа
4. Данные загружаются с API (не хардкод)
5. Pull Request создан и готов к ревью
