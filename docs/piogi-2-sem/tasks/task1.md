# Лабораторная работа 1

## Краткое описание

Настройка рабочего окружения: создание репозитория на GitHub, настройка `.gitignore`, первый коммит.

!!! warning "Блокирующая лаба"
    Без сдачи этой работы доступ к остальным лабораторным **закрыт**.

---

## Обязательные требования

### 1. GitHub-аккаунт

[ ] Создан аккаунт на GitHub (если не было)

### 2. Репозиторий
- [ ] Создан **приватный** репозиторий с названием `CinemaKiosk_Фамилия`
- [ ] Репозиторий содержит WPF Solution

### 3. Файл .gitignore
- [ ] В корне репозитория есть файл `.gitignore`
- [ ] Настроен для Visual Studio (исключает `bin/`, `obj/`, `.vs/`)

### 4. README.md
- [ ] В корне репозитория есть файл `README.md`
- [ ] Содержит ФИО и номер группы

### 5. Приглашение преподавателя
- [ ] Преподаватель добавлен в Collaborators

---

## Критерии оценивания

| Ранг | Требования |
|------|------------|
| **SILVER** | Репозиторий создан, код запушен, преподаватель приглашён. Возможны мелкие недочёты (мусор в репо — прощаем один раз) |
| **GOLD** | Идеальный `.gitignore`, чистый репозиторий, осмысленный коммит |
| **PLATINUM** | Красивый `README.md` с описанием проекта и чеклистом задач |

---

## Технические ограничения

!!! danger "Автоматическая пересдача"
    Наличие в репозитории папок `/bin`, `/obj` или `.vs` = пересдача.

---

## Критерий сдачи

Преподаватель получает приглашение в Collaborators, заходит в репозиторий и видит:

1. Чистый код без бинарников
2. Файл `.gitignore`
3. Файл `README.md` с данными студента
