# Лабораторная работа 5

## Краткое описание

Стилизация приложения: вынос стилей в ресурсы, создание ResourceDictionary, реализация переключения тем.

---

## Обязательные требования

### 1. ResourceDictionary
- [ ] Создан файл стилей (например, `Styles.xaml`)
- [ ] Подключён в `App.xaml` через `MergedDictionaries`

### 2. Стили для кнопок
- [ ] Создан стиль для основной кнопки (Primary)
- [ ] Кнопки имеют скруглённые углы (`CornerRadius`)
- [ ] Есть hover-эффект (смена цвета при наведении)

### 3. Цветовые ресурсы
- [ ] Цвета вынесены как ресурсы (`SolidColorBrush`)
- [ ] Используются `{StaticResource}` или `{DynamicResource}`

### 4. Темы оформления
- [ ] Созданы файлы `LightTheme.xaml` и `DarkTheme.xaml`
- [ ] Реализовано переключение между темами (кнопка или toggle)
- [ ] При переключении меняется фон и цвета текста

### 5. Нет inline-стилей
- [ ] В XAML-элементах нет `Background="Red"` и подобного
- [ ] Все стили через `Style="{StaticResource ...}"`

---

## Технические требования

### Структура файлов стилей

```
CinemaKiosk/
├── App.xaml
├── Themes/
│   ├── Styles.xaml
│   ├── LightTheme.xaml
│   └── DarkTheme.xaml
```

### Минимальный набор стилей

1. **PrimaryButton** — основная кнопка действия
2. **SecondaryButton** — второстепенная кнопка
3. **WindowStyle** — стиль окна с фоном
4. **CardStyle** — стиль для карточек

### Цветовая палитра (пример)

| Элемент | Light Theme | Dark Theme |
|---------|-------------|------------|
| Background | #FFFFFF | #1E1E1E |
| Card | #F5F5F5 | #2D2D2D |
| Primary | #007ACC | #007ACC |
| Text | #333333 | #FFFFFF |

---

## Критерий сдачи

1. Приложение запускается
2. Все стили в ResourceDictionary (не inline)
3. Кнопки имеют стилизацию (скругления, hover)
4. Работает переключение светлая/тёмная тема
5. Приложение выглядит как продукт, а не прототип
6. Pull Request создан и готов к ревью
